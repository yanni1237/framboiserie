Images du site La Framboiserie
==============================

Placez ici vos propres images avec exactement ces noms :

  hero.jpg          → Bannière en haut de page (framboises fraîches)
  barquette-100g.jpg → Carte "L'Individuelle (100g)"
  barquette-250g.jpg → Carte "La Classique (250g)"
  barquette-500g.jpg → Carte "La Familiale (500g)"

Formats acceptés : JPG, PNG ou WEBP (conservez le bon nom de fichier).

Si un fichier manque, l'image ne s'affichera pas dans le navigateur.
